## 2. 実装する要素（波紋のレイヤー）

- **要素:**
  - 単一の円ではなく、**複数層（例: 3枚〜4枚程度）重ねた円形の`div`要素群**。
  - これらを「波紋（Ripple）」のレイヤーとして扱う。
- **色:** 赤っぽい色（例: `bg-red-500/80`, `bg-rose-400/60` など）。奥のレイヤーほど透明度を少し下げておくと奥行きが出る。
- **配置:**
  - 全ての円をロゴの**背面中心**に絶対配置（`absolute`, center）で重ね合わせる。
  - 初期状態では、全てロゴの後ろに隠れている小さなサイズ。

## 3. アニメーションの挙動（拍動と波動）

ページ読み込み時、ロゴの背後から、心臓の鼓動に合わせて「波紋が次々と広がっていく」アニメーションを実行する。

- **リズムとタイミング（Heartbeat Rhythm）:**
  - 前述の「爆発的な開始と粘りのある余韻」（`Power4.easeOut` や `Back.easeOut`）をベースにする。

- **波動の表現（Ripple Effect / Stagger）:**
  - 重ねた複数の円を同時に動かすのではなく、**わずかな時間差（Stagger / Delay）**を設けて順番に発火させる。
  - **動きの流れ:**
    1.  一番上の円が「ドン！」と拍動して広がり始める。
    2.  0.1〜0.2秒遅れて、2枚目の円が同じ動きで追いかける。
    3.  さらに遅れて3枚目が続く。
  - これにより、単発の拡大ではなく、エネルギーが波及していく様子を表現する。

- **最終的な状態:**
  - 全ての波紋レイヤーが画面外まで広がりきり、かつ透明（Opacity: 0）になって消える。

## 4. 追加要素：TextTypeコンポーネントによるLoading表示

オープニング演出中、画面最下部に `@react-bits/TextType` コンポーネントを使用したタイピングアニメーションを配置する。

- **配置とスタイル:**
  - **配置:** 画面最下部中央（`absolute bottom-10 left-1/2 -translate-x-1/2`）。
  - **スタイル:**
    - フォント: 等幅フォント（Monospace）推奨。
    - サイズ: `text-xs` (極小)。
    - 文字間隔: `tracking-[0.25em]` (非常に広く)。
    - 色: `text-gray-400` や `text-white/60`（背景に馴染む色）。
    - 大文字: `uppercase`（小文字より幾何学的でモダンに見える）。

- **TextTypeコンポーネントの設定（Props）:**
  - **`text`:** `["LOADING...", "INITIALIZING..."]`
    - 複数の単語を切り替えることで、システムが稼働している感を演出。
  - **`typingSpeed`:** `120` ms
    - 意図的に遅く設定し、心臓の鼓動のような「落ち着き」を持たせる。
  - **`deletingSpeed`:** `60` ms
  - **`pauseDuration`:** `2000` ms
    - テキストが表示された状態で長く留め、せわしない印象を避ける。
  - **`cursorCharacter`:** `"|"` または `"_"`
  - **`cursorBlinkDuration`:** `0.8` s（ゆっくり点滅）。

- **全体のアニメーション同期（Sync）:**
  - この `<TextType />` をラップする親 `div` に対して、ロゴと同様の**「明滅（Breathing）」**アニメーション（Opacity 0.4 <-> 1.0）を適用する。
  - これにより、「文字がタイプされながら、全体がドックンと明滅する」という複合的な演出にする。
